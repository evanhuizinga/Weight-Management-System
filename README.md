# Weight-Management-System
Written in Python and runs off a CrowPi. Schematic image uploaded in schematic.png. Weight readings come from a strain gauge.
LEDs light independently light up when logs are empty, full, or when weight threshold is surpassed.
Switches will print store the current weight and time of day into a log, a switch that prints all logs currently in memory, and a switch that erases all logs currently in memory.
